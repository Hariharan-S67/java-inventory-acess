# Inventory Management System

This is a simple Java console-based application to manage products in an inventory.  
The program allows users to add and display product details using basic Object-Oriented Programming concepts.

## Features
- Add product details
- Display product information
- Simple and easy-to-use console system

## Technologies Used
- Java
- OOP (Object-Oriented Programming)

## How to Run

1. Compile the program:
   javac Main.java

2. Run the program:
   java Main

Make sure all Java files are in the same folder before running the program.
